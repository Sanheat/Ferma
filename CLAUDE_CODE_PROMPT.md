## Правила качества

- **Один файл на страницу** — все брейкпоинты внутри через @media
- **Нет дублирования** — токены и компоненты только в общих файлах
- **Pixel-perfect на 1440px** — это основной брейкпоинт
- **Работающие hover/focus states** — берёшь из ui_states вариантов
- **Русские тексты** — используй placeholder-тексты из Figma как есть
- **Изображения** — замени на CSS-заглушки нужного размера с background: #f0f0f0

1. Создай `tokens.css`
2. Создай `components.css`
